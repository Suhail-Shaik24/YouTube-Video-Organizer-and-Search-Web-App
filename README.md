# YouTube Video Organizer and Search Web App

## Overview

This project is a web-based application designed to help users organize YouTube videos efficiently. Users can add YouTube video links, categorize them using custom categories and subcategories, and later search and retrieve videos quickly. The application leverages the YouTube Data API to fetch video metadata and provides a clean, searchable interface for easy access.

## Features

* Add YouTube video links
* Automatically fetch video details using YouTube Data API
* Assign custom categories and subcategories to videos
* Search videos by title, category, or subcategory
* Organized and user-friendly interface
* Fast retrieval of saved videos

## Technologies Used

* Frontend: React
* Language: JavaScript
* API: YouTube Data API
* Styling: Tailwind CSS

## Project Workflow

1. User enters a YouTube video link
2. Video metadata is fetched using the YouTube Data API
3. User assigns a category and subcategory to the video
4. Video details are stored in the application state or database
5. User searches videos using keywords, categories, or subcategories
6. Matching videos are displayed for easy access

## How to Run the Project

1. Clone the repository
   git clone [https://github.com/your-username/youtube-video-organizer.git](https://github.com/your-username/youtube-video-organizer.git)
   cd youtube-video-organizer

2. Install dependencies
   npm install

3. Configure YouTube API key

   * Create a project in Google Cloud Console
   * Enable YouTube Data API v3
   * Add your API key in the project configuration

4. Run the application
   npm start

5. Open in browser
   [http://localhost:3000](http://localhost:3000)
